# Expense Tracker CLI
A simple command-line expense tracking application built with C#.

## Project Source

This project is based on the roadmap.sh challenge:
https://roadmap.sh/projects/expense-tracker

## Features
- Add expense
- Update expense
- Delete expense
- View all expenses
- Total summary
- Monthly summary

## Tech Stack
- C#
- .NET
- JSON file storage

## How to Run
dotnet run
